Overview

This experiment focuses on data preprocessing techniques, specifically data binning and data formatting. These techniques are essential for transforming raw data into a structured and meaningful form suitable for analysis.

Two datasets are used to demonstrate binning and formatting operations effectively.

Tools and Technologies Used
Python
Pandas
NumPy
Objectives
To understand data binning techniques
To categorise continuous data into discrete groups
To perform data formatting and transformation
To improve data readability and usability
Dataset Description
Dataset 1: Product Data

Contains information about products:

Product
Price
Units Sold
Dataset 2: Food Delivery Orders

Contains delivery-related information:

Order ID
Order Value
Delivery Time
Distance (km)
Techniques Implemented
Part A: Data Binning
1. Price Binning
Categorised price into: Low, Medium, High
Helps in grouping products based on cost
2. Units Sold Binning
Categorised sales into: Less Sell, Moderate Sell, High Sell
Useful for analysing product demand
3. Order Value Binning
Categorised order values into: Low, Medium, High
Helps identify customer spending patterns
4. Delivery Time Binning
Categorised delivery time into: Fast, Medium, Slow
Useful for performance analysis
Part B: Data Formatting
1. Data Type Conversion
Converted numerical columns to appropriate data types
Example: Integer to Float
2. Text Formatting
Converted text data to uppercase
Ensures consistency in categorical values
3. Rounding Values
Applied rounding to numerical data
Improves readability
4. Sorting Data
Sorted dataset based on specific columns
Helps in ranking and comparison
5. Unique Values Identification
Extracted unique categories from columns
Useful for understanding data distribution
6. Category Count
Counted frequency of categories
Helps in analysing distribution
Key Insights
Data binning simplifies continuous data into meaningful groups
Formatting ensures consistency and improves data quality
Categorisation helps in better interpretation and decision-making
Sorting and counting provide useful summary insights
Conclusion

This experiment demonstrates how data binning and formatting techniques can transform raw data into structured and interpretable formats. These preprocessing steps are essential for efficient data analysis and modelling.
